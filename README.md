# SCOPF

## Name
Security Constrained Optimal Power Flow

## Description
A solver of SCOPF-problems. It can take input on the Matpower-format, both as a csv-file and an xlsx-file. Constraints can be set manualy and automatically.

## Installation
Julia 1.7.2
