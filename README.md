# SCOPF

## Name
Security Constrained Optimal Power Flow

## Description
A solver of SCOPF-problems. It can take input on the Matpower-format, a csv-file or an xlsx-file. Constraints can be set manually and automatically.

## Installation
Julia 1.10

## Usage
Everything and nothing can work when running this code. It is only testet and developed during my PhD on a single computer.
